Red5 is an Open Source Flash Server written in Java.

http://code.google.com/p/red5/

Requirements:

- Java 1.6 or later

Features:

- Streaming Video (FLV, F4V, MP4)
- Streaming Audio (MP3, F4A, M4A)
- Recording Client Streams (FLV only)
- Shared Objects
- Live Stream Publishing
- Remoting
